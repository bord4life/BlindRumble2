# BlindRumble


This mod completely changes the graphics to a echolocation system. Now, you use sound to see things.
With this, you won't always see your opponent, so you have to predict and stay ahead of them at all times! As sound has a delay in real life, there is a delay before you can see your opponent or structures.

## Main Features

- Changes visuals to a sonar/sound system
- Players create snapshots whenever they do a pose or uses any form of movement.
- Seismic slam - Creates a vibration that reveals everything on the map for you and your opponent.
<hr>

## Requirements
This mod only triggers when one or more of these conditions are fulfilled:
- Both players have the mod enabled
- User is in Friends-only queue
- User is in a public/private park (Host doesn't matter)
